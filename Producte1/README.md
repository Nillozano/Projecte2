Aquest producte combina dues eines essencials per al treball professional dins la consultora **EverPia**:

1. La **gestió visual de tasques amb metodologia Kanban** mitjançant Microsoft Planner.
2. La **creació i ús d’un repositori personal a GitHub**, que serà el vostre espai de treball individual i de documentació tècnica.

Amb aquestes dues parts aconseguirem que cada grup i cada alumne tingui un sistema d’organització clar, transparent i professional, igual que qualsevol equip real del sector IT.
