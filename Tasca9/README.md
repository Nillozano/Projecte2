En aquesta tasca **individual**, cada alumne haurà de dur a terme una **investigació pròpia** per localitzar un servei de **hosting gratuït** que permeti fer proves.

L’objectiu és **experimentar amb un entorn real de servidor web** i disposar d’un espai on poder fer pràctiques.

Un cop escollit el proveïdor, caldrà **crear un compte, activar el hosting i instal·lar un WordPress funcional.**
